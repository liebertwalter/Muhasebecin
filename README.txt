# Muhasebecin

Bu paket yerel çalışan sürümdür.

İçindekiler:
- index.html
- style.css
- app.js

Özellikler:
- Borç ekleme / silme
- Gelir, gider, fatura, maaş kayıtları
- Aylık / yıllık özet
- Grafik
- Bildirim paneli
- Tema geçişi
- JSON yedek alma / yükleme
- Basit sesli komut desteği

Kurulum:
1. Dosyaları GitHub repo içine yükle
2. Eski index.html, style.css, app.js yerine bunları koy
3. Commit changes yap
4. GitHub Pages linkini aç
